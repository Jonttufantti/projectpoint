**Role:** Front-end & Full-stack developer (UI lead, integration with back-end)

ProjectPoint is a Trello-like web application built for the City of Turku, designed to manage departments, projects, boards, task cards, and users. The app streamlines project workflows, task assignment, and departmental management.

---

## Features

### Front-end (React / TypeScript / Vite)
- Fully responsive UI with modular components.
- Dynamic dashboards for projects, boards, and task cards.
- Modals for creating/editing projects, boards, tasks, and users.
- Attachments support with file previews and deletion.
- Filtering and sorting of users, tasks, and departments.
- Navbar with dynamic URLs, department-specific tables, and active/inactive user handling.
- Playwright test automation for login, project management, and core flows.

### Back-end (Django / Django REST Framework / JWT)
- RESTful APIs for user, project, board, and task management.
- JWT authentication and password recovery flows.
- Department-based access control.
- Data seeding and admin management for rapid testing.
- Secure handling of attachments and API endpoints.
